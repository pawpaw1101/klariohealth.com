# Klario Website Structure

Klario is organized into two layers:

1. Public website
   - `index.html` - Home, prototype screenshots, and feature snapshots
   - `about.html` - Who we are, how we started, the problem, and the solution
   - `features.html` - Product features and user-facing benefits
   - `login.html` - Try for free entry point

2. Logged-in web app
   - `app/dashboard.html` - Health workspace overview
   - `app/documents.html` - Uploaded reports and documents
   - `app/upload.html` - Report upload options
   - `app/timeline.html` - Longitudinal health history
   - `app/trends.html` - Biomarker trend charts
   - `app/family.html` - Family and pet profiles
   - `app/account.html` - Profile, security, and account controls
   - `app/settings.html` - App preferences

The public navbar has `[logo]`, About, and Features aligned left. The Try for free button is aligned right and leads to the login page, which then leads into the web app prototype.
